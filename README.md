# brackeys_2026_1: "Careful"

This is our code for the game submission of Brackey's Game Jam 2026.1.
It is a barebones game where you can navigate three different mazes, while facing a bunch of different enemies.

It was generated using [libGDX](https://libgdx.com/).


The game submission for the game jam is available at https://sleepdeprivedduo.itch.io/careful.
